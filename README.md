## Usage
### Install
```sh
git clone https://github.com/Fox-Boss/Telegram-Fake-Views
```
### Compile and run
```sh
go build .
./Fox_booster -h
```
For example: `./Fox_booster -l=https://t.me/F0X_Boss-p=proxies.txt -w=1100`
Recommended workers count(-w) - 500. Maximum workers count - 1100
